Nama Aplikasi : Gigih Spotify
Deskripsi : Aplikasi mirip Spotify untuk Create Playlist dari lagu yang anda inginkan
Cara install: 
1. pergi ke link gigih-spotify.vercel.app
2. Klik untuk authorize akun spotify kamu

Cara menjalankan:
1. Setelah login , kamu berada di halaman create playlist.
2. Di Create Playlist kamu bisa mencari lagu yang ingin kamu masukkan kedalam playlist mu
3. Add To Playlist
